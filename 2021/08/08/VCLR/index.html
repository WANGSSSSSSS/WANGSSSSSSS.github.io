Video Contrastive Learning with Global Context [NCE-LOSS]

this article is mainly contribute the following:

  • global context sampling method for video level contrastive learning
  • a video consensus constrain for self supervised learning

model structure

the inter loss uses augmented samples P1+P_1^+P1+​​ from anchor frame Q1aQ_1^aQ1a​ ​and samples {P2,P3}\{P_2, P_3\}{P2​,P3​}​​ from the same video to formulate positive

Intra-Loss

when using the intra loss, the samples from the same video will also be discrimination as negative.

  • anchor tuple ta={v1,v2,..vk},vi∈Sit^a = \{ v_1, v_2, .. v_k\}, v_i \in S_ita={v1​,v2​,..vk​},vi​∈Si​

    t+={v1+,v2+,..vk+},vi+∈Sit^+ = \{ v_1^+, v_2^+, .. v_k^+\}, v_i^+ \in S_it+={v1+​,v2+​,..vk+​},vi+​∈Si​
