Spatial Transformer Networks

spatial transformer networks boils down to three main components :

Localization network

localization network is a regular CNN regresses the transformation parameters, the transformation is not learned explicitly via a supervise procedure, instead the six parameters output latent space is forced by achieve a optimal accuracy.

Grid Generator & Sampler

Grid generator generates a grid sampler tensor using the rigid affine transformation, this grid tells where will the output pixel from, generally bilinear interpolation applied.
