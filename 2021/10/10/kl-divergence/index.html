一个关于分布度量的话题

感觉数据的分布是一个很值得说的东西，在看VAE, VRNN, GAN 这些生成模型的论文的时候，总结的一个基本点就是:

生成模型要捕捉到数据的分布，然后使得输出和输入的相似度最高

$ min_{P_\theta} Dis(\hat p(x),\hat q_\theta(x) )$

很自然的，损失函数，监督信号就会定义在如何衡量两个分布之间的相似度上，最常见的分布对比

直接定义两个分布对应概率差距，也就是在密度函数上进行差异的估计

连续情形： $loss = \int [P(x) - Q(x)]^2 dx$

离散表示： $loss = \Sigma [P_i - Q_i]^2 $

这是一种简单的策略，但是在训练过程中，非线性效果不够好,可能是其他的原因，就比如简单的分类任务上就抛弃了这个损失，这更像一个回归的监督方式，在目标检测回归包围框时使用

在信息论中，衡量两个分布之间的相似度，可以通过相对熵，或者叫KL 散度来衡量

$$H(p||q) = p(x) log({p(x) \over q(x)}) = -p(x)log(q(x)) + p(x)log(p(x)) = H(p,q) - H(p)$$

连续情形：$ loss = \int p(x) log({p(x) \over q(x)}) dx$

离散表示：$loss = \Sigma p_i log({p_i \over q_i})$

对于一个多分类任务，我希望输入向量经过神经网络之后的类别分布，符合类别的信息，什么意思呢？

对于样本${\vec x, i}$, 表示该样本为第$i$类，那么他的期望类别分布$p(x)_k$, 其中$ 1 \leq k \leq N$, 表示类别下表，期望输出应该是这样的一个离散分布$p(x)$,：

$$p(x) = {C_1 (0),C_2(0),….C_i(1), …C_N(0) }$$

对应的，网路输出的类别分布，则可能是另外一种离散分布$q(x)$：

$$q(x) = {C_1,C_2,….C_i, …C_N }$$

把这个分布带入到，离散表示的KL散度中去：

$$loss = -log(q_i)$$

这个式子就是在似然估计上大名鼎鼎的 NLL(negative log likelihood )

对于$q(x)$分布，可以增加更多的非线性，或者说为了整流归一化，引入Softmax

$$\hat q(x) = softmax(q(x)) = { {e^{Ci} \over \Sigma_1^N e^{C_k}} }, \ \ i\in [1,N]$$

$$loss = -log(\hat q(x)_i) = C_i + log[\Sigma_1^N e^{C_k}]$$

这个就是 交叉熵损失 CrossEntropy， 当然没有考虑权重，权重是在minibatch中起作用的，用于调节随机梯度的主方向，也就是更需要优化哪个分类

所以，可以得到一个结论， 在目标离散分布为冲击型的情况下，交叉熵损失等价于KL散度损失

还有些问题

分布的度量监督，更适合于离散分布，两个相邻维度之间没有关系，但是如果维度的下标有了意义，基于期望的监督是不是也有意义了？
