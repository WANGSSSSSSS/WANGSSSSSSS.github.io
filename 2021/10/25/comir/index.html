CoMIR: Contrastive Multimodal Image Representation for Registration

multimodal image contrastive learning is the basic topic in machine learning and computer vision, this paper proposes a general contrastive method to use neural network mapping a different modal to a mono-modal, which is a latent space of original modal. Thus some operations like different modal image registration or image segmentation can be very efficient.

this paper mainly argues the following two viewpoints, which are also their pivotal contribution

Rotational equivariance

The equivariance is sounds like invariance, which is an orthodox idea, while the equivariance is infrequent.

Given a modality independent function map F:x→yF : x \to yF:x→y, TxT^xTx is the inverse transformation for origin space corresponding to TyT^yTy, which is applied for latent space. The equivariance can be formulated as the following:

​ $F(T^x(x))= T^y(F(x)) $

this viewpoint is pivotal in this contrastive approach, instead of conventional self-supervise in computer vision, which is only based on instance discriminator theory

contrastive learning

this paper use Unet map the original input pair x1→x2x_1 \to x_2x1​→x2​, which are different modal input pair, to a mono modal, use self-supervise methods to force y1,y2y_1, y_2y1​,y2​ in a same latent space $y_n \in \Omega $

​ yn=UNetn(xn)y_n = UNet_n(x_n)yn​=UNetn​(xn​)

as mentioned in the previous sector, this contrastive learning will not only supervise which feature map y2iy_2^iy2i​instance is corresponding to the given feature map y1y_1y1​, but maximize the inner product between the inverse transformed feature map. TTT is an general affine transform for 2d image.

​ max h(F(T1xx1),F(T2xx2))   x1→x2max \ h(F(T^x_1x_1), F(T^x_2x_2)) \ \ \ x_1 \to x_2max h(F(T1x​x1​),F(T2x​x2​))   x1​→x2​

​ $max \ h(T_2^y(y_2), T_1^y(y_1)) $

this operation is a joint supervise action aims to force the latent feature map yyy is equivariance aspect to rotation. if we separate the joint action, we will get the equal constrains:

​ max [h(T1y(F(T1x(x)),F(x))]max \ [ h(T_1^y(F(T_1^x(x)), F(x))]max [h(T1y​(F(T1x​(x)),F(x))]
